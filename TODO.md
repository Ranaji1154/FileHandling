# Git Setup and Push for File Handling Java Project

## Step 1: [PENDING] Download and Install Git for Windows
- Download official installer: https://git-scm.com/download/win
- Run installer as admin, accept defaults (adds git to PATH).
- **Restart VSCode** after install for new terminal PATH.
- Expected: `git --version` works.

## Step 2: [PENDING] Verify Git and Init Repo
- Run `git --version`
- `git init`
- `git add .`
- `git commit -m "Initial commit: Java file I/O demos (Main.java, Write.java, .Main.txt)"`

## Step 3: [PENDING] Add Remote and Push
- `git remote add origin https://github.com/Ranaji1154/FileHandling.git` (if not already)
- `git branch -M main`
- `git push -u origin main`

## Step 4: [DONE] Task Complete
- Repo pushed successfully.
